Please double-click "StartGRINEdit.jar" to start GRINEdit.
It will take about 10 seconds to run at first time.

"StartGRINEdit.jar" executes "grinedit-app-alpha0.20.jar"
in same folder. So you can run this directly.In the case, messages are printed to standard output
so you should better to run from console.

StartGRINEdit.jarをダブルクリックして起動します。
初回起動時には多少時間がかかります(10秒程度)

StartGRINEdit.jarは同一フォルダ内の
grinedit-app-alpha0.20.jarを起動するだけのプログラムなので、
こちらを直接起動しても構いません。
その際は、エラーメッセージが標準エラー出力に出るので
コンソールから起動することをおすすめします。
